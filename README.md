## Introduction
CHASE DB1 dataset
CHASE-DB1 is a retinal vessel segmentation dataset consisting of 28 color retinal images with a size of 999 Ã— 960 pixels, collected from the left and right eyes of 14 students. Each image is annotated by two independent human experts.
